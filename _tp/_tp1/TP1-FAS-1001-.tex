\hypertarget{premiuxe8re-uxe9tape}{%
\subsection{Première étape}\label{premiuxe8re-uxe9tape}}

\includegraphics{images/Screen Shot 2024-01-16 at 6.09.49 PM.png}

La première étape était d'utiliser la commande `cd' qui permet de
manipuler les dossiers. Mes dossiers se trouvent sur mon desktop et j'ai
utilisé `ls' pour vérifier le nom de mon dossier nécessaire. Ensuite
j'ai tenté d'utiliser le lien SSH mais ça dit que je n'ai pas les
``correct access rights''. Alors, j'ai essayé le lien HTTPS ce qui a
fonctionner, puis avec la commande `git clone' j'ai pu copier les
fichiers sur mon desktop.

\hypertarget{commande-git}{%
\subsection{Commande git}\label{commande-git}}

\includegraphics{images/Screen Shot 2024-01-20 at 5.57.34 PM.png}

J'ai tenté de faire la commande git commit. Cependant, de ce que j'ai
compris, je devais relier mon terminal à mon compte github. Ensuite, la
commande git commit ne fonctionnait pas donc sur internet j'ai vu
l'option que je pouvais utiliser ``-m'' pour ajouter un message a mon
commit.

\includegraphics{images/Screen Shot 2024-01-20 at 6.09.09 PM-01.png}
